Instrucciones:
        Para ejecutar nuestro programa se debe utilizar el siguiente comando en la 
        terminal de Python:

                            python tarea1AC.py input1.txt input2.txt

        donde input1.txt es un archivo con la entrada para la pregunta 1 (Join en Bases de Datos)
        e input2.txt es el archivo que tiene la entrada para la pregunta 2 (Contando Triangulos en Grafos).

Aclaraciones:
        Nuestro programa durante su funcionamiento crea el archivo de texto llamado formato.txt, el cual se crea 
        para facilitar la ejecucion del programa.
